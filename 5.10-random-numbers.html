5.10 Random Numbers

Notes

Random Numbers are useful in software development for many reasons- game mechanics, generating usernames/passwords, anytime you need something seemingly unpredictable.

Pseudo-Random Numbers- computers are just big calculators solving formulas and cant create truly random numbers, but instead use an algorith that generates numbers that seem random.

Random Numbers in JS

Javascript provides a number of functions that are useful for generating random numbers

  • Math.random() is a function that returns a pseudo-random number beetween 0 and 1not uncluding 1
  • Math.floor() is a function thatr takes a decimal number and rounds it down - chops off the decimal part

Random number Calculations

To produce a random integer from a min value to a max value (including the min and the max) use this calculation

math.floor(Math.random() * (max + 1)) + min

Example

Genrate a random number from 1 to 10 and store it in a variable

var randNum = Math.floor(Math.random() * 11) + 1>
